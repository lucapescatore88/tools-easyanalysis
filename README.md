# Easyanalysis

To checkout the project
```bash
git clone ssh://git@gitlab.cern.ch:7999/LHCb-RD/tools-easyanalysis.git
```

To setup the proper environment
```bash
source scripts/setup.sh (or setup.csh if you use tcsh)
```

To build the libraries
```bash
make
```
