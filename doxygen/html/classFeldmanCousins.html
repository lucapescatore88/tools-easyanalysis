Main function to extract the pvalues grid after cunstructing and initializing the object.

Parameters
pointsYou can give a set of custom points to scan. If this is empty a regular grid is generated.
origParsOriginal parameters. This is needed in the case you did some fitting before and the current parameters of the PDF are not/ the initial ones. This is important because each fit to data and toys has to begin with the same initial parameters to be compared. If this is empty it copied the current parameters of the PDF at the initialization moment.
pointReturns the point of best fit
opt: Options
isValid
nll_valAllows to set an external value for the min LL of data. Can reduce execution time. mostly for internal use.
