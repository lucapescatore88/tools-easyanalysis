Detailed Description

This class can be used to do simultaneous fits to more than 1 sample (in 1D)
Can also be used fitting different variables for each sample.
Finally nothing forbids to use the class to fit "simultaneously" different variables of the same sample. But just be aware that in this case it works well only if the PDFs of the various 1D fits are uncorrelated (namely PDF in N-d = prod(2D PFDs))
