Parameters
brContains a list of branch name (also partial names)
optDefines what to do with the "br" names list
opt = "names" -> keeps only branches with exact names in br
opt = "contains" -> keeps branches with names or partial names in br
opt = "except" -> keeps all branches whose exact names are not in br
opt = "exceptcontains" -> keeps all branches whose names don't contain (even partially) names in br
